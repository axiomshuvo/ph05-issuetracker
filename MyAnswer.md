# 1️⃣ What is the difference between var, let, and const?

-- var - var is old and function-scoped, so it can be accessed outside blocks; it can also be redeclared and reassigned
--Let - let is block-scoped, so it cannot be accessed outside blocks; it cannot be redeclared but can be reassigned
-- Const - const is block-scoped, so it cannot be accessed outside blocks; it cannot be redeclared or reassigned and must be initialized

# 2️⃣ What is the spread operator (...)?

-- Spread operator used to expand elements of array or object, helps copy, merge, and pass values easily

# 3️⃣ What is the difference between map(), filter(), and forEach()?

-- Map - used to loop through an array and always returns a new array , original array remains unchanged

-- filter() - used to loop through an array and return a new array based on condition. Only elements that match the condition are included , original array remain unchanged

-- forEach() - used to loop through an array and perform an action on each element, does not return anything , Original array remain unchanged

# 4️⃣ What is an arrow function?

-- Shorter function syntax using => , automatically returns value in single line not multiple lines , doest not have its own this , used for cleaner code

# 5️⃣ What are template literals?

-- Using backticks (``) to create dynamic strings, allows variables with ${} and supports multi-line strings
